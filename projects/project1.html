Project Overview

This project is a full-stack e-commerce application built with the MEAN stack (MongoDB, Express.js, Angular, Node.js). It allows users to browse products, view details, add items to a shopping cart, and place orders. Users can create accounts, log in securely, and manage their profiles. The admin dashboard enables product management (add, update, delete), order tracking, and user management. The system provides a responsive design, secure authentication, and a smooth shopping experience.

Technologies Used

Angular Node.js MongoDB Express.js
